raytracer-rs

A raytracing renderer created with the Rust programming language. Based on the books/guides found in the Raytracing in One Weekend book series.

Features:

  • Multithreading
  • JSON parsing support for scene creation
  • Bounding Volume Hierarchy
  • Textures
  • Diffuse Lighting
  • Point Lights
  • Volumes
  • Shape Instancing
  • Reflections
  • Dielectrics/Refraction
